Demonstrasi Tag Media & Konten Sematan

Halaman ini menampilkan contoh penggunaan berbagai tag media di HTML.

1. Tag <img>

Digunakan untuk menyematkan gambar ke dalam halaman web.

2. Tag <audio>

Digunakan untuk menyematkan konten audio. Atribut 'controls' menampilkan kontrol pemutar audio.

3. Tag <video> dan <source>

Digunakan untuk menyematkan konten video. Tag <source> di dalamnya dapat menentukan beberapa format file video.

4. Tag <iframe>

Membuat sebuah bingkai (inline frame) untuk menampilkan dokumen HTML lain di dalam halaman saat ini. Contoh: Peta Google.

5. Tag <embed>

Cara sederhana untuk menyematkan konten eksternal seperti PDF atau SVG (dukungan browser bisa bervariasi).

6. Tag <object>

Cara yang lebih standar dan kuat untuk menyematkan objek. Bisa digunakan untuk gambar, video, atau bahkan halaman lain.

7. Tag <svg>

Memungkinkan Anda untuk mendefinisikan grafis berbasis vektor langsung di dalam kode HTML. Tidak pecah saat diperbesar.

8. Tag <canvas>

Hanya menyediakan sebuah "kanvas" atau area kosong. Untuk menggambar di atasnya (garis, bentuk, gambar), Anda **wajib** menggunakan JavaScript.

Browser Anda tidak mendukung elemen canvas.

(Area di atas kosong karena tidak ada JavaScript yang dijalankan untuk menggambar.)
